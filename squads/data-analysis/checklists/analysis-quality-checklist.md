# Checklist: Analysis Quality

## Rigor
- [ ] Business question restated and answerability defined
- [ ] Data profiled on the actual data provided (no assumed shape)
- [ ] Data-quality issues flagged with impact on conclusions
- [ ] Descriptive stats computed (not estimated by eye) for relevant variables
- [ ] Patterns reported with confounders named; correlation ≠ causation respected

## Integrity
- [ ] No fabricated values, rows, or distributions
- [ ] Every insight tagged with confidence + evidence
- [ ] Limits of the data and analysis stated explicitly

## Output
- [ ] Report conforms to analysis-report template; lifecycle = draft
- [ ] 3–7 decision-relevant insights present
