# Checklist: Value Equation & Grand Slam Offer

## Value Equation
- [ ] Dream outcome is the customer's true end state (not a feature)
- [ ] Perceived likelihood backed by real proof/guarantee (gaps marked, not faked)
- [ ] Time delay minimized or reframed (speed-to-first-win identified)
- [ ] Effort & sacrifice minimized (DFY/DWY/tooling where it raises value)
- [ ] Value Score computed and compared across ≥1 variant

## Grand Slam Components
- [ ] Every major obstacle has a corresponding deliverable
- [ ] Delivery vehicles chosen with value vs. cost-to-deliver tradeoff explicit
- [ ] Each bonus names the specific objection it dissolves
- [ ] Guarantee is the strongest defensible type, terms stated precisely
- [ ] Scarcity/urgency mechanisms are real and enforceable (no fake countdowns)
- [ ] Offer name is benefit-led and specific (not clever-but-vague)

## Integrity
- [ ] No fabricated testimonials, numbers, or proof
- [ ] All gaps marked `# gap: needs proof`
- [ ] Output conforms to offerbook-entry template; lifecycle = draft
